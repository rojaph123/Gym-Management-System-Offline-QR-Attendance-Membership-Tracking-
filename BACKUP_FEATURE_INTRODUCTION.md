## Common Questions

**Q: Does this cost anything?**
A: No! Backup and restore are completely free.

**Q: What if my backup file gets lost?**
A: Create another backup. Keep multiple copies in different places.

**Q: Can I restore on a different device?**
A: Yes! Backup on one device, restore on any other device.

**Q: How long does a backup take?**
A: Usually less than 30 seconds, even for large gyms.

**Q: How long does restore take?**
A: 2-5 minutes depending on how much data you have.

**Q: Is my data encrypted?**
A: Your backup file is JSON (not encrypted by default), but cloud services like Google Drive provide encryption.

**Q: What about my PIN?**
A: PIN is not backed up for security reasons. You'll create a new PIN on any restored device.
